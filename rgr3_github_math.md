## 1. Построение моделей

### 1.1. Линейная модель

```math
\hat y = a + bx.
```

```math
\hat a = -2655.84807, \qquad \hat b = 96.85392.
```

Итоговое уравнение:

```math
\hat y = -2655.84807 + 96.85392x.
```

### 1.2. Квадратичная модель

```math
\hat y = a + bx + cx^2.
```

```math
\hat a = -7745.24079, \qquad \hat b = 261.47953, \qquad \hat c = -1.29548.
```

Итоговое уравнение:

```math
\hat y = -7745.24079 + 261.47953x - 1.29548x^2.
```

### 1.3. Степенная модель

```math
\hat y = ax^b.
```

Линеаризация:

```math
\ln y = \ln a + b \ln x.
```

```math
\hat a = 0.69423, \qquad \hat b = 2.03444.
```

Итоговое уравнение:

```math
\hat y = 0.69423x^{2.03444}.
```

## 5. Итоговый вывод

Исследована зависимость числа поездок от температуры по данным файла `RGR3_D_1.csv`. Построены линейная, квадратичная и степенная модели. Все модели показывают положительную связь между температурой и числом поездок. По критериям $R^2$, RSS и RMSE лучшей является квадратичная модель: $R^2 = 0.47533$, RSS = 91612550.34711, RMSE = 1070.12003. Степенная модель имеет наименьшую среднюю ошибку аппроксимации $A = 32.06476\%$, но уступает квадратичной по остальным показателям. Для линейной модели коэффициент при $x$ статистически значим: $t_{	ext{набл}} = 8.25762$, $p$-value $= 2.980e-12$. Для прогноза выбрана квадратичная модель, по которой при $x^* = 67.7863$ ожидаемое число поездок равно 4026.78791.
